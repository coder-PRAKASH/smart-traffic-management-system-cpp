# 🚦 Smart Traffic Management System using C++

## 📌 Project Overview

The Smart Traffic Management System is a simulation-based project developed using C++. The project is designed to manage traffic flow efficiently, reduce congestion, and prioritize emergency vehicles using queue scheduling and object-oriented programming concepts.

This project demonstrates real-world traffic handling logic using data structures such as Queue and Priority Queue.

# 🚗 How the System Works

1. Traffic signal timings are initialized.
2. Vehicles are added to traffic lanes.
3. Normal vehicles are stored in a queue.
4. Emergency vehicles are stored in a priority queue.
5. Emergency vehicles are processed first.
6. Remaining vehicles are processed in FIFO order.

---

# 🧠 Data Structures Used

## Queue

Used for processing normal vehicles in FIFO order.

## Priority Queue

Used for prioritizing emergency vehicles such as ambulances and fire trucks.
